Get Started

Onepager.js will turn your markup into a onepage scrollable application with a dynamic navigation, API, callbacks and a mobile version.
                  
                     npm install onepager.js
                     import OnePager from 'onepager.js'
                     const yourInstance = new OnePager()
                     import './node_modules/onepager.js/dist/css/onepager.min.css'

Markup

Feel free to start your onepage application with this markup
                  
                     <header class="onepager-header">                                             
                        <nav class="onepager-navigation-container"></nav>                        
                        <div class="onepager-navigation-mobile-opener-container"></div>                     
                     </header>                     
                     <main class="onepager-wrapper">                        
                        <div id="panel-1" class="onepager-panel" data-onepager-title="Panel 1">                           
                           <p>Panel 1 content</p>                        
                        </div>                        
                        <div id="panel-2" class="onepager-panel" data-onepager-title="Panel 2">                           
                           <p>Panel 1 content</p>                        
                        </div>                        
                        <div id="panel-3" class="onepager-panel" data-onepager-title="Panel 3">                           
                           <p>Panel 1 content</p>                        
                        </div>                     
                     </main>

Option

Description

Default value
contentWrapper The content of the onepager. .onepager-wrapper
panelsSelector Selects the onepager items/panels. .onepager-panel
navContainer Onepager will inject the navigation into this container .onepager-navigation-container
navMobileOpenerContainer Onepager will inject the navigation opener icon into this container .onepager-navigation-mobile-opener-container
headerContainer This container will become the fixed header .onepager-header
activePanel The active panel will be marked with this css class active
activeNavItem The active navigation item will be marked with this css class active
inViewOffset By default, onepager considers a panel active if it breaks any edge of the viewport by at least 150 pixels. (can be a positive or negative integer). Onepager uses in-view.js find more infos about offset here { top: 150, bottom: 150 }
navMobileOpenerIcon

Onepager will inject this icon into the navMobileOpenerContainer Image like this <img src="navMobileOpenerIcon" /> it will open the navigation on the mobile version (onClick).

navItemAnimation After opening the mobile navigation the items will be animated in. You can choose between: fadeInRightShort, fadeInLeftShort, growIn, fadeInRight. Onepager uses http://jackonthe.net/css3animateit/examples/ if you need other animations add the additional css. fadeInRight
animateContent Coming soon! false
scrollSpeed The speed of the scroll event 600
mobileBreakpoint The breakpoint in "px" from which the mobile navigation will be activated 786
onScrollStart A callback function that will kick in before the scroll event fires after using a navigation item undefined
onScrollEnd A callback function that will kick in after the scroll event ends after using a navigation item undefined
onEnter(panel) A callback function that will kick in after a panel becomes visible. The function takes an argument "panel" and it's the object of the panel that is becoming visible undefined

API

Method

Description
const myOnePager = new OnePager() Instanciates your onepager
myOnePager.scrollTo('#unique-selector') Scroll to a panel via unique selector
myOnePager.scrollToTopOfCurrentPanel() Scroll to the top of the current panel
myOnePager.navMobileOpen() On Viewports smaller than the defined mobile breakpoint opens the mobile navigation
myOnePager.navMobileClose() On Viewports smaller than the defined mobile breakpoint closes the mobile navigation
myOnePager.navMobileIsOpened() Check if the mobile navigation is opened (returns boolean)
myOnePager.currentPanelInView() Returns the current panel object

Callbacks

                  
                     const onePager = new OnePager({
                        onScrollStart: (event) => {
                           console.log(event)
                        },
                        onScrollStop: (event) => {
                           console.log(event)
                        },
                        onEnter: (el) => {
                           console.log(el)
                        }
                     })

Styling

Styles

Comments

Code
Active Navigation Element

override this if you need another active navigation item style

 

                              .onepager-desktop .onepager-navigation-container li a.active {
                                 color: aqua;
                              }
Header Styles

the header (mobile & desktop) as default behaves in a flex manner. You will have to override this if you need a more complex navigation

 

                              .onepager-header {
                                 display: flex;
                                 justify-content: space-between;
                              }
Panel Styles

the panels (mobile & desktop) as default behave in a flex manner. The content will be vertically and horizontally centered. You will have to override this:

 

                                 .onepager-panel {
                                    display: flex;
                                    align-items: center;
                                    justify-content: center;
                                 }
Web & Desktop

Onepager follows the mobile first styling concept. A body class .desktop depending on the defined breakingpoint will be added to override the mobile styles

 

                                 body.desktop {
                                    // add your desktop styles here
                                 }
